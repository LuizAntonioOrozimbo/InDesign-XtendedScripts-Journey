# Cabeçalhos padrão (.jsx)

Esta pasta contém **modelos de cabeçalho** para scripts ExtendScript utilizados ao longo do projeto **InDesign ExtendScript Journey**.

Os arquivos aqui servem como **base inicial** para aulas, testes e projetos, garantindo consistência e organização do código.

---

## Como usar

1. Copie o arquivo de cabeçalho desejado
2. Renomeie o script conforme o contexto
3. Atualize os campos de descrição, versão e data
4. Implemente a lógica do script dentro da função `main()`

---

## Observação

Os cabeçalhos não são aulas completas.  
Eles existem para padronizar estrutura, comentários e organização dos scripts.
