# Estrutura base de aulas

Esta pasta contém um **template padrão** para a criação de aulas do projeto **InDesign ExtendScript Journey**.

O objetivo é manter consistência na organização, clareza didática e facilidade de leitura ao longo do curso.

---

## Como usar

1. Copie o arquivo `estrutura-aula.md`
2. Cole na pasta da aula correspondente ao módulo
3. Renomeie conforme o padrão do curso
4. Ajuste as seções conforme a complexidade do conteúdo

---

## Observação

Nem todas as aulas precisam conter todas as seções do template.  
Use apenas o que fizer sentido para o tema abordado.
